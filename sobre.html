Sobre o projeto

Este projeto foi desenvolvido como proposta para a disciplina de Projeto Integrador 3 da Universidade Virtual do Estado de São Paulo (UNIVESP), nos cursos do eixo de computação, compreendendo Tecnologia da Informação, Ciência de Dados e Engenharia da Computação.

O trabalho atende aos critérios estabelecidos pela disciplina, que compreendem:
Desenvolver um software com framework web ou aplicativo que utilize banco de dados, inclua script web (Javascript), nuvem, acessibilidade, controle de versão, integração contínua e testes. Incluir um dos seguintes requisitos: uso e fornecimento de API, análises de dados e IoT
